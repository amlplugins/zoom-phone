# @amlplugins/zoom-phone

Native `@zoom/rest-api` SDK re-exported under the `@amlplugins` namespace.

> Zoom Phone REST API — call queues, users, phone numbers, call logs, voicemail, SMS.

## Install

```bash
npm install @amlplugins/zoom-phone
```

## Use

```ts
import { /* named exports from @zoom/rest-api */ } from "@amlplugins/zoom-phone";
// or
import sdk from "@amlplugins/zoom-phone";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
